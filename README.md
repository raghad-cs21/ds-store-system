# Store Management System (Data Structures Project)

This project is a simple store management system implemented using core data structures concepts in C++.

## Features
- Product management
- Order processing
- Data storage using data structures
- Console-based user interaction

## Technologies Used
- C++
- Data Structures
- Object-oriented programming basics

## Purpose
The project was developed as part of a Data Structures course to practice implementing real-world systems using fundamental data structures.
